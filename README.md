* Redux 
* Tailwind CSS

## Installation 

1. `git clone https://github.com/Waterbottle88/cat-pinterest.git`
2. `cd cat-pinterest`
3. `npm install`
4. `npm start`
5. Open https://localhost:3000 in your browser
